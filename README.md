### Prerequisites
- Node.js (v18+)
- Java JDK (v17+)

### 1. Backend Setup
1. Navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Build the project:
   ```bash
   ./mvnw clean package  # or 'mvn clean package' if installed globally
   ```
3. Run the application:
   ```bash
   ./mvnw spring-boot:run # or 'mvn spring-boot:run'
   ```
   The backend will start on `http://localhost:8080`.

### 2. Frontend Setup
1. Open a new terminal and navigate to the root folder:
   ```bash
   cd .. # Back to root if you are in backend
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
   The frontend will start on `http://localhost:5173`.

## üì¶ Deployment

2. Set up a PostgreSQL/MySQL database service.
3. Configure `application.properties` with environment variables for DB credentials.
4. Deploy using the Dockerfile or Buildpack.

### Frontend (Vercel / Netlify)
1. Push the `src` and root config files to your repo.
2. Set `Output Directory` to `dist`.
3. Set `Build Command` to `npm run build`.
4. Add environment variables for the backend URL if needed.
